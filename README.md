# ai-Consultation-System
An AI-based online consultation system that simulates the doctor consultation process through natural language interaction, helping users preliminarily analyze health issues.

# AI智能医疗诊断系统

一个基于Web的AI辅助医疗诊断系统，通过多轮对话问诊，为患者提供初步诊断建议和治疗方案。

## 功能特点

- **六步完整流程**：免责声明 → 基本信息 → 症状选择 → AI问诊 → 诊断报告 → 导出
- **智能症状选择**：支持多身体部位、多症状选择
- **AI多轮问诊**：3-5轮智能对话，深入了解病情
- **完整诊断报告**：包含主要诊断、置信度评估、严重程度、鉴别诊断、病理分析、药物治疗方案、治疗建议、警示信号
- **报告导出**：支持文本和JSON格式导出

## 快速开始

### 1. 环境要求

- Python 3.7+
- pip

### 2. 安装步骤

#### Linux/Mac:
```bash
cd ai_medical_system
chmod +x start.sh
./start.sh
```

#### Windows:
```cmd
cd ai_medical_system
start.bat
```

#### .env密钥修改：
```
以文本编辑模式进入.env
OPENROUTER_API_KEY=你的密钥
OPENROUTER_URL=密钥提供商的网址
```

#### 手动安装:
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

### 3. 访问系统

打开浏览器，访问：http://localhost:5000

## 使用流程

### 第一步：免责声明
- 阅读重要提示
- 勾选"我已阅读并同意上述免责声明"
- 点击"同意并继续"

### 第二步：基本信息
- 填写姓名、性别、年龄
- 选择过敏史（可多选）
- 选择既往病史（可多选）
- 填写当前用药情况
- 选择生活习惯（吸烟史、饮酒史）

### 第三步：症状选择
- 选择出现症状的身体部位（可多选）
- 选择具体症状（可多选）
- 详细描述症状情况

### 第四步：AI智能问诊
- 与AI进行3-5轮对话
- 回答AI的问题
- 问诊完成后点击"生成诊断报告"

### 第五步：诊断报告
- 查看完整的AI诊断报告
- 包含：主要诊断、置信度、严重程度、鉴别诊断、病理分析、药物治疗方案、治疗建议、警示信号
- 可选择"重新开始"或"导出报告"

### 第六步：导出
- 选择导出格式（文本或JSON）
- 预览报告内容
- 自动下载报告文件

## 诊断报告内容

### 1. 主要诊断
- 根据症状和问诊信息得出的初步诊断

### 2. 置信度评估
- AI对诊断结果的置信度百分比
- 置信度理由说明

### 3. 严重程度
- 轻度/中度/重度评估
- 帮助患者了解病情紧急程度

### 4. 鉴别诊断
- 列出需要鉴别的其他疾病
- 说明鉴别要点

### 5. 病理分析
- 发病机制说明
- 支持点和不支持点分析

### 6. 药物治疗方案
- 药物名称
- 用法用量
- 医保类别
- 禁忌症

### 7. 治疗建议
- 非药物治疗建议
- 生活方式调整
- 复诊建议

### 8. 警示信号
- 需要立即就医的危险信号
- 帮助患者识别紧急情况

## 支持的疾病类型

系统目前支持以下常见症状的诊断：

- **头部症状**：头痛、头晕等
- **眼部症状**：视力模糊、眼睛干涩等
- **耳部症状**：耳鸣、听力下降等
- **鼻部症状**：鼻塞、流涕等
- **咽喉症状**：咽痛、咳嗽等
- **胸部症状**：胸痛、呼吸困难等
- **心脏症状**：心悸、心跳加速等
- **腹部症状**：腹痛、恶心、呕吐等
- **腰背症状**：腰痛、背痛等
- **四肢症状**：四肢疼痛、麻木等
- **关节症状**：关节疼痛、红肿等
- **皮肤症状**：皮疹、瘙痒等
- **精神心理症状**：焦虑、抑郁、失眠等
